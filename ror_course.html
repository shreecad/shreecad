Ruby on Rails...

Ruby on Rails(Rails)(RoR), is an open source web application framework for the Ruby programming language that closely follows the Model-View-Controller (MVC) architecture. This is mainly used for developing web application by faster, simpler and with less code and bugs by following Agile methodology and TDD.

In this course you will learn how to do web application development using Rails. Before that you will be trained about OOps concept, Basic Ruby language and MVC Architecture. So after finishing this course you will be having more knowledge and experience about Ruby language, Rails framework, TDD and Development of web application.
